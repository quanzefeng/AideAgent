# GoodAgent 可清理目录分析

## 结论

`desktop/` 目录是完全自包含的 Electron 桌面应用，不依赖父级的任何目录。以下内容可以安全删除，不会影响 GoodAgent 桌面应用运行。

## 可删除目录

| 目录 | 大小 | 说明 |
|------|------|------|
| `src/` | 34M | Claude Code 原始源码，与桌面应用无关 |
| `build-src/` | 34M | 构建时 transform 产生的中间产物 |
| `stubs/` | 8K | Bun 编译桩，构建 `src/` 用的 |
| `scripts/` | 48K | 构建 `src/` 用的脚本 |
| `tools/` | 9K | Claude Code 的自定义工具 |
| `types/` | 1K | Claude Code 的类型定义 |
| `utils/` | 3K | Claude Code 的工具函数 |
| `docs/` | 68K | 逆向分析文档，纯文档资料 |
| `node_modules/` | 406M | 根目录依赖，给 `src/` 构建用的 |

## 可删除文件

| 文件 | 说明 |
|------|------|
| `package.json` | 根目录的，给 `src/` 构建用的，非桌面应用 |
| `package-lock.json` | 根目录依赖锁文件 |
| `tsconfig.json` | TypeScript 配置，给 `src/` 构建用的 |
| `QUICKSTART.md` | Claude Code 构建说明 |
| `.gitignore` | 根目录的 git 忽略规则 |
| `GoodAgent.jpg` | 项目截图/封面图 |
| `goodagent-snapshot.yml` | 界面快照数据 |
| `preload_debug.txt` | 调试日志 |
| `stderr.txt` | 调试日志 |
| `stdout.txt` | 调试日志 |

## 不可删除

| 目录/文件 | 原因 |
|-----------|------|
| `desktop/` | GoodAgent 桌面应用本体 |
| `.git/` | Git 版本控制记录 |

## 注意事项

`desktop/package.json` 的 `electron-builder` 打包配置中引用了 `../dist`（Claude Code 构建产物），但该目录目前不存在，且运行时不需要。仅在打包发布 Electron 应用时才会用到，如需发布可重新构建。
